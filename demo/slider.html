Design Styles

Presets

Every slider below is the same component — only CSS custom properties differ. No variants, no modes, no code changes. Any design system's slider can be replicated by tuning variables alone.

All presets above are achieved by overriding --slider-track-h-*, --slider-thumb-radius, --slider-thumb-w-*, --slider-thumb-h-*, and surface tokens. No class changes needed.
