Design Styles

Presets

Every calendar below is the same component — only CSS custom properties differ. Material, Apple, Airbnb-range, enterprise — all one component.

All presets achieved by overriding --datepicker-day-radius, --datepicker-day-size-*, --datepicker-shadow, --datepicker-selected-bg and surface tokens. No class changes needed.

Accessibility

A11y

Focus ring on day cells, keyboard navigation (arrows, Home, End), disabled days, and forced-colors support.

Focus Ring

Tab into the grid. :focus-visible ring wraps each day cell.

Keyboard Nav

← → ↑ ↓ arrows move focus. Home = first day, End = last day.

Disabled Days

data-disabled reduces opacity, blocks interaction.

Forced Colors

High Contrast: Highlight, ButtonText, GrayText.
