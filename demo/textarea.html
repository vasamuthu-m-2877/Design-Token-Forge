Height Modes

Auto vs Fixed

Auto (default): textarea grows with content, text aligns top, rows attribute sets the initial height. Fixed (data-height="fixed"): textarea height is locked to the size-scale spacing token (same as button heights), text is vertically centered and truncates with ellipsis on overflow.
