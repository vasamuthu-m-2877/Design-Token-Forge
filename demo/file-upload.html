Slots & Anatomy

6 Sub-elements

Label, icon, text, CTA link, hint, helper, and file list. All slots are optional except the input.

Motion

3 Animations

Loading shimmer on CTA text, pulsing border on dragover, and enter animation. Reduced motion disables all.

Accessibility

A11y

Focus ring, disabled state, error validation, and forced colors support.

Focus Ring

Tab to the upload zone. :focus-visible shows a 2px keyboard-only outline. Outline width, color, and offset are tokenised.

Native Input

Hidden <input type="file"> covers the full zone. Screen readers announce file input natively. Label uses for attribute.

Forced Colors

High Contrast shows ButtonText border, Highlight on hover, GrayText when disabled, LinkText for CTA.

Reduced Motion

All transitions resolve to instant (0.01ms) when prefers-reduced-motion: reduce is active. Progress bars included.
