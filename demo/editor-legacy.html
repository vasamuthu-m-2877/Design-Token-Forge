1. Color Roles

All Palettes

Each role generates a WCAG-guaranteed 22-step palette from a single key color. Desaturated is auto-derived from the brand hue. Add custom roles for accent or domain-specific colors.

1b. Fixed & Utility Colors

4 Tokens · Theme-aware

Standard UI colors not tied to any role. Fixed tokens stay the same in both themes. Theme-aware tokens swap in dark mode.

T0 primitives feed into T1 semantic roles

2. T1 — Semantic Token Map

18 Tokens × Light/Dark

Each primitive step is assigned a semantic purpose. Select a role to see how its 22-step palette maps to 18 semantic tokens for both light and dark themes. Toggle Edit Mappings to customize step assignments.

T1 semantic tokens drive T2 surface contexts

2b. T2 — Surface Context Map

9 Surfaces × 16 Props

Each surface resolves its step names against a palette source. Select a surface to view and edit its step assignments — use the palette source dropdown to switch which role palette feeds the surface.
