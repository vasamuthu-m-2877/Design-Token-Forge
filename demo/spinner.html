Accessibility

A11y

Spinner uses role="status" for live-region announcements and supports prefers-reduced-motion.

Screen Reader Uses role="status" and aria-label="Loading" so assistive tech announces the loading state without interrupting the user.
Reduced Motion When prefers-reduced-motion: reduce is set, ring slows to 1.5s and dots freeze in a static staggered-opacity pattern.
Forced Colors In Windows High Contrast mode, arc uses Highlight and track uses GrayText for maximum visibility.
Disabled State data-disabled reduces opacity to 40% and removes pointer events. Use when loading is cancelled but element stays in DOM.
