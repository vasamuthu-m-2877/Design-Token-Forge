05 · Compose from Intent

Figma plugin AI compose. Describe → instantiate L1s on canvas → tweak freely.
← all wireframes

The future surface. Designer types intent; plugin returns real, editable Figma layers built from DTF L1 atoms bound to project tokens. Not a black-box export — the output is composed of components the designer can remix freely.

The promotion loop — used 3+ times → review

DTF Plugin · Promotion candidate detected pattern reuse
📌 Pattern detected
"Sign in form" composed 3 times in this project
Used in: /sign-in · /admin/auth · /docs/embed-auth
Promote this composition to a reusable pattern?
Promotion is opt-in. Most patterns become recipes (cheap). Only well-tested compositions graduate to L2 components. This is how DTF stays lean. 1
  • Promotion is the bridge between "designer's intuition" and "system canon." Three uses = a real signal. The plugin asks; doesn't enforce.
