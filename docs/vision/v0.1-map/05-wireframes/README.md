## Conventions

- `┌─┐` boxes for UI regions
- `▶` for primary actions, `▷` for secondary
- `[…]` for inputs, `(•)` for radio, `[x]` for checkbox
- Mermaid for flow / state diagrams
- Keep wireframes annotated — every box names *why* it's there
