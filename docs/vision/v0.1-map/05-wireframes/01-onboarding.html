Key decisions encoded in this flow

  • Brand color is the only required field. Everything else has a default.
  • Archetype is optional — designer can skip to a blank L1-only setup.
  • Output is Figma + code + CLI simultaneously — no lock-in to one surface.
  • Example L3 pages are scaffolds, never "the way it must look."
  • No login required to try this flow. Onboarding is the demo.
