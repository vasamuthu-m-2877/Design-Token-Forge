## How to review

Mark sections inline with `[KEEP]`, `[REWORK]`, `[EXPAND]`, `[CUT]`, or
`> COMMENT: ...`. Whole-section verdicts at the bottom of each file are most
useful. Don't worry about prose quality — react to *structure and direction*.
